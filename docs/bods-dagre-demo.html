What this shows: UNILEVER PLC (GLEIF) with 4 direct subsidiaries. Interest type: otherInfluenceOrControl, beneficialOwnershipOrControl: false. The BOVS entity icons are embedded as inline SVG so they work in both file:// and HTTP contexts.

Arrow direction (BOVS): Arrows point FROM the owner/controller (UNILEVER PLC, top) TO the controlled entity (subsidiaries, bottom), following the BOVS directionality guideline that arrows indicate the flow of ownership/control in the reading direction of the diagram.

Edge rendering fix: bods-dagre's Dr/Pr decorators hide the original dagre edge (opacity:0) and insert a BezierJS-offset clone. When the offset degenerates (empty PolyBezier or NaN), the clone is removed and the original is restored as a grey fallback. When the offset succeeds, only the styled clone shows — no duplicates.

BOVS icons fix: bods-dagre uses SVGInjector (XHR) to load icons, which is blocked by the browser in file:// context. This demo pre-inlines all BOVS SVG icons as a JavaScript lookup table, then intercepts the img.injectable elements after draw() to substitute inline SVG directly.

Single-pass edge fix in BODSGraph.tsx: Replaces the previous two-pass approach (which unconditionally restored all originals, causing duplicate visible edges).

// Single-pass: for each bods-dagre styled clone (.own/.control):
// – valid path → leave it (original stays hidden, clone provides BOVS styling)
// – invalid path (empty / NaN / too short) → remove clone, restore original
el.querySelectorAll('g.edgePath.own, g.edgePath.control').forEach(clone => {
  const path = clone.querySelector('path');
  const d = path?.getAttribute('d') ?? '';
  const isInvalid = !d || d.trim().length < 10 || d.includes('NaN');
  if (isInvalid) {
    const prev = clone.previousElementSibling;
    if (prev?.classList.contains('edgePath') &&
        !prev.classList.contains('own') && !prev.classList.contains('control')) {
      prev.style.opacity = '1';
      const p = prev.querySelector('path');
      if (p && !p.getAttribute('stroke') && !p.style.stroke) {
        p.style.stroke = '#666';
        p.style.strokeWidth = '1.5px';
        p.style.fill = 'none';
      }
    }
    clone.remove();
  }
});
